# Shopping cart demo

Using the foundation template (included) and angularjs.

## Requirements

  * Ruby 1.9+
  * [Node.js](http://nodejs.org)
  * [compass](http://compass-style.org/): `gem install compass`
  * [bower](http://bower.io): `npm install bower -g`
  * Angularjs 1.2+

## Use

 *Just  open index.html !
